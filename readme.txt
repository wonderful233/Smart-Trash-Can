# 智能垃圾桶系统
 STM32 + ESP32 + Python 图像识别的智能垃圾桶+微信小程序监控

## 项目功能
- 远程监控和控制开盖
- 图像识别垃圾分类
- 双单片机协同控制

本项目使用的云平台为巴法云平台，硬件设备为STM32f103c8t6、NodeMCU
STM32硬件接线如下
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
模块类型					垃圾桶 1（左桶）			垃圾桶 2（右桶）				备注
舵机（SG90）				PWM：PA8（TIM1_CH1）		PWM：PA11（TIM1_CH4）		舵机需 5V 供电
温湿度传感器（DHT11）		DATA：PB5				DATA：PB8					
满溢检测超声波（HC-SR04）	Trig：PB12/ Echo：PB6		Trig：PB14 / Echo：PB7			检测距离 0-20cm（满溢阈值：<10cm），VCC 接 5V、GND 共地
手动按键key控制开关			PA0						PA1							接地即可
串口（后续接 ESP8266）		TX：PA9（USART1_TX）		RX：PA10（USART1_RX）			波特率 115200，用于后续数据上传

NodeMCU与STM32的TX RX 反接即可 需共地！！！
